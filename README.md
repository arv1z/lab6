Для этого нужно добавить ветвление в конфигурационные файлы для CI
со следующей логикой:

если commit помечен тэгом, то необходимо собрать пакеты
(`DEB`, `RPM`, `WIX`, `DragNDrop`, ...)
и разместить их на сервисе GitHub.
